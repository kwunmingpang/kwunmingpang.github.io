Flash Card Web App

May, 2022

This project can be accessed in here. I built it while preparing for my Japanese N1 exam, because I need an application that can share the flashcard list between devices, since usually I study using my PC and I don't usually carry PC with me daily.

It is a web application which users can input their own word list and reinforce their memory by scrolling in a virtual flashcard. Originally I use actual flashcards, but they are bulky and difficult to carry, and I was working at that time, I really want to find something that I can use while commuting to work easily, since it usually takes me about one hour to commute.
